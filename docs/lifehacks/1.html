СОСТАВ

PP (полипропилен) — это крышки для бутылок,
одноразовая посуда, упаковки пищевых
продуктов, стаканы для йогуртов, цветочные
кашпо, баночки из-под косметики,
упаковки для линз, капсулы от шоколадных яиц.

PP (полипропилен) — это крышки
для бутылок, одноразовая посуда,
упаковки пищевых продуктов, стаканы
для йогуртов, цветочные кашпо, баночки
из-под косметики, упаковки для линз,
капсулы от шоколадных яиц.

КАК СОРТИРОВАТЬ?

  • Не выбрасывать.
  • Сдавать вместе с тюбиком.
  • Если на крышке есть маркировка с указанием вида
    пластика, положить в соответствующую категорию.
